# 自由入力の例（20件）

これらは作業者が実際に入力する可能性のある自然な文章です。標準用語の表記ゆれを含んでいます。

## 例1: モーター異常音 → ベアリング劣化 → ベアリング交換
```
プレス01のモーターから変な音がしている。確認したところベアリングが摩耗しているようだ。ベアリングを新品に交換して、グリスも補充した。動作確認OK。
```

## 例2: 油漏れ → シール劣化 → シール交換
```
油圧ユニットからオイル漏れを発見。シール部を確認するとOリングが劣化していた。パッキンを交換し、増し締めを実施。漏れ止まりを確認。
```

## 例3: センサーエラー → 配線断線 → 配線修復
```
コンベア03の位置センサーがエラー表示。配線をチェックしたらケーブルが断線していた。配線を交換して結線やり直し。導通確認して復旧完了。
```

## 例4: ベルト滑り → ベルト張力不足 → ベルト張力調整
```
送風機のVベルトがすべって回転が伝わらない。確認したところベルトが緩んでいた。テンション調整して適正な張りに設定。正常動作を確認した。
```

## 例5: 過熱アラーム → 冷却不足 → 冷却系点検清掃
```
制御盤から高温警報が出た。見るとフィルターが詰まって通風不良になっていた。フィルター交換とファン清掃を実施。温度が正常範囲に戻った。
```

## 例6: 振動異常 → 軸ズレ → 芯出し調整
```
ポンプの振動がひどくガタついている。点検したら芯出し不良でカップリングがズレていた。アライメント調整して軸心を合わせた。振動が収まった。
```

## 例7: 圧力低下 → ポンプ劣化 → 予防保全記録
```
油圧回路の圧が上がらず低圧警報が出ている。油圧ポンプの吐出量を測定したら性能低下していた。ポンプ交換が必要だが、部品手配中のため後日対応予定。
```

## 例8: モーター異常音 + 振動 → 複合原因
```
モータがうるさくて振れも大きい。調べたらベアリングがダメになっていて、さらにボルトも緩んでいた。ベアリング取替とボルト締め直しを実施。
```

## 例9: 油漏れ + 圧力低下 → 複合症状
```
シリンダから油がにじんでいて、しかも圧力も出ない。シール部から漏れているのとポンプも調子悪い。とりあえずOリング交換したが、ポンプも要点検。
```

## 例10: センサー汚れ → センサー清掃
```
センサー反応なしのエラー。見たらセンサーが汚れていただけだった。清掃して位置も微調整したら正常に戻った。簡単な作業で済んだ。
```

## 例11: ベルト摩耗 → ベルト交換
```
ベルトが空転するので確認したら、ベルト自体が摩耗してすり減っていた。張り調整してもダメだったのでベルトを新品に交換。プーリーも清掃した。
```

## 例12: 配線接続不良 → 端子締め直し
```
センサーが時々エラーになる。配線をチェックしたら端子が緩んでいた。端子を締め直して絶縁テープで補強。その後エラーは出ていない。
```

## 例13: 冷却ファン故障 → ファン交換
```
温度上昇警報が頻発。調べたら冷却ファンが故障して回っていなかった。ファンを交換して温度センサーも確認。正常動作に戻った。
```

## 例14: 軸受損傷 → 緊急停止対応
```
回転中に異音発生して緊急停止。軸受けが損傷していて焼き付き寸前だった。応急処置でグリス補充したが、至急ベアリング交換が必要。
```

## 例15: 油圧ホース劣化 → ホース交換
```
配管接続部から油が漏れている。ホースを確認したら劣化してひび割れていた。油圧ホースを交換してフィッティング部も増し締め。漏れ止まり確認。
```

## 例16: 制御基板異常 → 基板交換
```
制御パネルが動かない。調査したら制御基板の部品が焼損していた。予備基板に交換して設定を復元。動作確認して運転再開。
```

## 例17: チェーン伸び → チェーン張力調整
```
チェーンコンベアの動きが悪い。チェーンが伸びて弛んでいた。テンショナーで張りを調整。チェーンの注油も実施して正常運転に戻した。
```

## 例18: グリス不足 → グリスアップ
```
軸受部から異音。グリスが切れていた。グリスアップを実施して音は止まった。定期的な給脂が必要と判断。次回点検時にも確認する。
```

## 例19: エアー圧力不足 → エアフィルター交換
```
シリンダの動きが遅い。エア圧を確認したら低下していた。エアフィルターが詰まっていたので交換。ドレン抜きも実施。圧力が回復した。
```

## 例20: 複合トラブル → 総合点検
```
ラインAのプレス機でトラブル続き。モーター音がおかしいし、油も少し漏れている。全体的に点検したらベアリング劣化とシール不良を発見。両方とも部品交換して、ボルト類も全数確認。総合的な整備を実施した。
```
